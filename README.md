# ZorLore

ZorLore is an MMORPG builder for creators who want to design, run, and ship their own online world without stitching together a server, editor, client, and asset pipeline by hand.

![ZorLore Screenshot](https://zorlore.com/assets/zorlore-screenshot.jpg)

With ZorLore, you build the game world itself: the map, cities, monsters, NPCs, quests, spells, items, actions, visuals, and progression. The platform is built around a browser-playable Tibia-style MMORPG stack, paired with a world editor that lets you control the details instead of fighting the toolchain.

## What ZorLore Includes

- A game server built around original 7.6 mechanics
- A browser client at [zorlore.com](https://zorlore.com/) with no game download required
- The ZorLore editor for building and maintaining your world
- A bridge/database layer for account, world, and runtime integration

## Game Server Features

- Original 7.6 mechanics
- Configurable experience rate, regeneration rate, skill rate, and loot rate
- Lua scripting throughout the game
- Script your own quests, NPCs, spells, and monsters
- Run a world that you can tailor mechanically and content-wise to your own design

## Build Your Own World

ZorLore is not just a server. It is a full MMORPG creation workflow.

Using the ZorLore editor, you can:

- Edit every aspect of the game world
- Build maps, cities, depots, houses, monster spawns, raids, and actions
- Create and tune items, objects, readable text, liquid states, and visuals
- Edit sprites directly inside the editor
- Define how your game looks, feels, and plays without leaving the toolchain
